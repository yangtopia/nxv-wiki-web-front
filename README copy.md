# nxv-wiki-web-front

Nexivil Archive Labs v1.0.0

## 개발환경

- Node.js 12.13.0 이상
- Next.js 9.3.0 이상 (latest)
- React.js 16.13.0 이상

### 패키지 설치

```bash
> npm install
```

## 개발

### Local (Development)

```bash
> npm run dev
```

### Local (Production)

```bash
> npm run dev:prod
```

### 빌드

```bash
> npm run build
```

## 배포

GITHUB CI/CD
